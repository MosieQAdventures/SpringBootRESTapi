Learned basic usage of SpringBoot and MySQL in a week. Mini-App created in around 2 weeks.
I was using Postman app to send Get/Put/Post/Delete requests, but it works using cmd too.
App provides 3 necessary endpoints described in 'Task Goal.txt' + some basic crud ones.

Changed command creation to enable request sends (' ' for '_' / '^' for '.').
Send command using: http://localhost:9000/newjob/{command}

More details below.
________________________________________

? HELP:
?
? Command creation is possible by using below format:
? amount-/int/_min-/int/_max-/int/_chars-/chars/
? 
? where:
? amount - number of strings to be created
? min - minimal length of the created string
? max - maximum length of the created string
? chars - set of character to create strings from
? /int/ - integer value
? /chars/ - characters from keyboard
? 
? Use '.' (dot) sign with 1 '_' sign on both sides to separate multiple commands: "_._"
? 
? Example commands above.
